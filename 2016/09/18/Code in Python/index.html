  • List Comprehension

    It has many uses and rules. We can change a comprehension to a longer but simpler code to understand it. For example:

  • list.append(x)

    This method can add a new element at the end of the list.

  • list.extend(x)

    In this method, x is a list. It can add x to the end of the list.

  • list.remove(x)

    Remove the first element whose value is x in the list.

  • list.pop(x)

  • If x is blank, then it return the last element of the list and delete it.

  • Or x should be a number of place of the element you want to return and delete.

  • Other methods of list

  • list.remove(x): remove the first item whose value is x.
  • list.clear(): remove all the items.
  • list.index(x): return the index of the first item who has value as x.
  • list.count(x): return the number of times x appears.
  • list.sort(): sort the list.
  • list.reverse(): reverse the list.
  • list.copy(): return a copy of the list.

Tuple

  • Define a new tuple

    Use parentheses to define a new tuple(can also ellipsis). But if you want to define a tuple that only has one element, you need to type a comma after the element.

Personally speaking, I think if you don’t care about the safety of your data, tuples are useless. Well, I know this thought is naive.

Set

A set is an unordered collection with no duplicate elements.

  • Define a new set

    Use {} or set() to create a new set. If you want to create an empty set, use set(), because {} will create a new empty dictionary.

Deque

Deque can insert or delete an item from both ends and is faster than list. Before use a deque, you have to import it from collections.deque:

  • deque.appendleft(x)

    Add x to the front of the deque.

  • deque.remove(x)

    Remove the first item whose value is x.

  • deque.extendleft(deq)

    This is a little strange because when add items in deq to deque, it will reverse the order.

  • deque.remove(x)
