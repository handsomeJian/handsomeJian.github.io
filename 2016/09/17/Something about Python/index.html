  • Import a Module

For example, you want to import a module called ‘hehe’ and use the function ‘haha’ in it, how to do this?

If you type import hehe , then when you should use ‘haha’ in a way like : hehe.haha()

You can type from hehe import haha, so you can directly use haha() in your program.

You can also type from hehe import * to import all the functions in the module.

You can use dir(hehe) to list all the functions in hehe

PS: If you directly use $ python hehe.py , the __name__ of the module will be changed to "__main__" .
